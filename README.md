# Features available:
1. Search for any movie using searchbar.
2. Click the heart icon to add a movie to favorites.
3. Favorite tab on the toolbar to quickly access favorite movies.

**NOTE: If testing on emulator, please use the device keyboard to submit search. Hitting enter on keyboard has undefined behavior.**


# Libraries used.
1. Room DB for percistant storage.
2. Kotlin extentions for Android components.
3. Moshi for JSON parsing.
4. Retrofit for API calls.
5. Glide for loading images from thumbnail.
6. Kotlin coroutines for background work.
